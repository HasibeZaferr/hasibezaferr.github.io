Paycell Payment Technology

Paycell is a mobile application that offers financial solutions such as bill payment, online shopping, money transfer, OR transactions. There are 20Mn+ users on Android, iOS, Huawei platforms.


Paycell Mobile Application


Development with Android Java & Kotlin

I worked as an Android Developer in the Paycell project for about 3 years.

Paycell project was a very comprehensive project written in Android Java. When I started working on the project, I developed with Java. About 1 year later, it was decided to convert the project from Java to Kotlin. We converted the project as Android team.


Paycell Know Your Customer

In Paycell, I worked very excited project Known Your Customer. I have a Udacity Machine Learning certificate so this artificial intelligence project was assigned to me. Known Your Customer project is a customer identification project with artificial intelligence consisting of 3 steps. I developed the end-to-end integration of this project to the Paycell mobile application. We received a patent award with this project.

This was a very important development for the project. It was featured on some news sites as it was a pioneering project in Turkey.


Paycell Huawei Integration

We were using Google Firebase services in the Paycell project. After Huawei Mobile Services was released, I developed all Huawei integrations for the project to be published on the Huawei Store.


Paycell Big Family

Paycell team was a big family for me. After working for about 3 years in Paycell, I took a break from corporate life for to experience the entrepreneurial adventure.
